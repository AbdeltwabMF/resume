# Resume

Single-page, one-column resume. It uses standard LaTeX packages and custom commands to keep formatting consistent and easy to edit.

## Preview

![RESUME PREVIEW](assets/preview.png)

## Requirements

- LaTeX with `pdflatex`
- `latexmk`
- `make`
- `poppler`

## Build

```bash
make all clean
```
